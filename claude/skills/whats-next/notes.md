# whats-next — maintainer notes

This skill is the canonical home for the session-start resume protocol.
markdown.md's "When resuming work from a tasks file" subsection points here
instead of duplicating the steps — single source of truth.

The skill is meant to fire on natural-language "what's next" / "what should I
work on" at session start, not only on an explicit `/whats-next`. Its
description carries those trigger phrases so the match happens from the
always-loaded skill list, without needing markdown.md loaded first.
